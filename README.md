# SynchroBots_PLC

## 구성도
시스템 아키텍처 및 통신 흐름도 / 설계


## 담당 역할 
- AMR, ARM, PLC 등 서로 다른 장비를 OPC UA 표준으로 통합하는 서버 설계 및 개발
- 설비 상태 및 제어 명령을 객체 지향적 노드 구조로 설계하여 데이터 교환 안정성 확보
- XG5000 기반 컨베이어 로직 설계 및 Modbus RTU 통신을 통한 PC 연동 제어 구현
- 프로젝트 각 파트 규격에 맞춘 하드웨어(AMR 적재함, 컨베이어 등) SOLIDWORKS 모델링

## 기술 스택
Python, XG5000, Modbus RTU/TCP, OPCUA, SOLIDWORKS
